# NU Admission Result 2026

> National University Honours Admission Result portal (Session 2025–2026)

## 🚀 Deploy to Netlify

### Step 1 — Upload to GitHub
1. Create a new GitHub repository (e.g. `nu-result-2026`)
2. Upload all files:
   - `index.html`
   - `style.css`
   - `app.js`
   - `favicon.svg`
   - `_redirects` *(important for Netlify routing)*

### Step 2 — Deploy on Netlify
1. Go to [netlify.com](https://netlify.com) → **Add new site** → **Import an existing project**
2. Connect your GitHub account and select the repository
3. Build settings:
   - **Build command**: *(leave blank)*
   - **Publish directory**: `.` (root)
4. Click **Deploy site**

Your site will be live at `https://your-site-name.netlify.app`

## ✨ Features

- Live result search → redirects to official NU website
- SMS format generator with copy & send buttons
- All merit list info (1st, 2nd, Migration, Quota, Release Slip)
- Step-by-step admission guide
- FAQ section
- Fully responsive (mobile-friendly)
- Animated counters and progress bars
